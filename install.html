Installation¶

Known to work on with Python 2.7 on modern Linux and OS X systems. Not tested on anything else.

One environment variable must be set:

  • SCRATCH - the name of a directory to download weather data to.

Relies on Iris, so first install that.

Uses the ECMWF api: ecmwfapi - to download CERA-20C and ERA5 data. Follow the instructions on that page to get an ECMWF api account (free). This is not required for 20CR data.

Then download or fork the source in https://github.com/philip-brohan/IRData:

  • Install with ‘python setup.py install –user’.
  • Test with ‘python setup.py test’.
